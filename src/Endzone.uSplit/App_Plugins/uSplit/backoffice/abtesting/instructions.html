uSplit - A/B Testing for Umbraco

If you are reading these instructions, you have already successfully installed uSplit. For reference, the installation instructions can be found here: http://endzonesoftware.com/installing-usplit/.

uSplit integrates with Google Analytics Experiments to provide your site with A/B testing. Apart from authenticating with Google Analytics on the dashboard, you need to include the Google Analytics tracking code inside your master template to allow uSplit to report experiment data back to Google.

Additional information about usage and common scenarios can be found in the user guide.
